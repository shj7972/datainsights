# Data Insights

A multi-app dashboard built with Streamlit for data analysis and visualization.

## Apps Included

- Korean Stocks
- Foreign Stocks
- Bonds
- Materials
- Cryptocurrency
- Forex
- Real Estate
- MBTI Matcher
- Basking Robbins 31 Game

## Local Development

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deployment

This app is configured for deployment on Vercel. The main entry point is `app.py`.
